# India AQI Analysis Dashboard 🌫️

## Project Overview
An in-depth analysis of Air Quality Index (AQI) 
across major Indian cities using Python and Power BI.

## Dataset
Downloaded from Kaggle:
https://www.kaggle.com/datasets/rohanrao/air-quality-data-in-india

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI

## Key Findings
- Analyzed 29,531 records
- Covered 26 Indian cities
- Data from 2015-2020
- Analyzed pollutants: PM2.5, PM10, NO2, SO2

## Visualizations
- Top 10 Most Polluted Cities
- AQI Trend Over Years
- PM2.5 Analysis by City
- Seasonal AQI Analysis
- Pollutants Heatmap
- Delhi vs Mumbai Comparison
